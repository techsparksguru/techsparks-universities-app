# Universities-frontend

Universities app frontend

## Where do I change the API_URL?

`For Local Dev`

```
edit .env
```

`For Docker Compose Up`

```
edit .env.docker
```

`For Production`

```
edit .env.production
```

And rebuild the image

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
